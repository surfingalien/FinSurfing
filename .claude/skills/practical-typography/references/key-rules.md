# Summary of Key Rules

Source: [Butterick's Practical Typography](https://practicaltypography.com/)

## Body Text (Most Important)

The four most important typographic considerations for body text are **point size**, **line spacing**, **line length**, and **font** because those choices determine how the body text looks.

### Point Size
- **Print:** 10–12 points
- **Web:** 15–25 pixels

### Line Spacing
- **120–145%** of the point size

### Line Length
- Average **45–90 characters** per line (including spaces)

### Font Choice
- Use a **professional font** (see font recommendations)
- **Avoid:**
  - Goofy fonts
  - Monospaced fonts (except for code)
  - Most free fonts
  - System fonts—especially Times New Roman and Arial

## Typography in Ten Minutes

These five rules make you better than 90% of writers and designers:

1. **Body text quality** determines document quality (there's more of it than anything else)
2. **Point size:** 10–12 pt (print) or 15–25 px (web)
3. **Line spacing:** 120–145% of point size
4. **Line length:** 45–90 characters average
5. **Professional fonts:** Buy one, never use Times New Roman or Arial

## Formatting Rules

### Quotation Marks
- ✅ Use **curly quotation marks**
- ❌ Not straight ones

### Emphasis
- Use **bold or italic** as little as possible
- **Never use both together**
- **Never underline** (except for web links)

### Capitalization
- **ALL CAPS** are fine for less than one line of text
- Use **5–12% extra letterspacing** with ALL CAPS and small caps

### Spacing
- Put **one space between sentences** (not two)
- Don't use multiple word spaces in a row
- Use **first-line indents** (1–4× point size) **OR** space between paragraphs (4–10 pt)—not both

### Alignment
- Use **centered text** sparingly
- Always use **hyphenation** with justified text

### Punctuation
- **Kerning** should always be on
- Don't confuse **hyphens and dashes** (—, –, -)
- Use **ampersands** (&) sparingly unless in proper names
- Use proper **trademark and copyright symbols** (™, ®, ©)
- In documents >3 pages, **one exclamation point is plenty**
- **Apostrophes** point downward (')
- **Ellipses** use proper character (…), not periods and spaces
- **Foot and inch marks** are straight (", '), not curly

### Special Characters
- Put **nonbreaking space** after paragraph and section marks (§, ¶)
- If you don't have real **small caps**, don't use them at all

## Common Mistakes to Avoid

1. **Times New Roman or Arial** — Only used by the apathetic and sloppy
2. **Underlining** (except links) — Typewriter habit
3. **Two spaces after periods** — Typewriter habit
4. **Fake small caps** (just shrinking regular caps)
5. **Multiple hyphens as dash** — Use proper em dash (—)
6. **Alphabetic approximations** of symbols — Use real ™, ©, §, etc.
7. **Mixing indent + spacing** between paragraphs — Pick one
8. **Justified text without hyphenation** — Creates bad spacing
9. **Overuse of exclamation points!!!**
10. **Bold + italic together** — Pick one

## Web-Specific

- Line length: prevent text from flowing to browser window edges
- Line spacing: use `line-height` property with unitless value (e.g., `1.3` for 130%)
- Point size: 15–25 pixels for comfortable reading

## Print-Specific

- Page margins: larger than one inch (typewriter habit to avoid)
- Point size: 10–12 points for body text
- Line spacing: use "Exact" option in word processors (not single/1.5-line defaults)
