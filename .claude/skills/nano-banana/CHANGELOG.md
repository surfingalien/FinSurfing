# Changelog

All notable changes to the Nano Banana skill.

## [2.0.0] - 2026-03-07

### Changed
- Updated default model from `gemini-3-pro-image-preview` to `gemini-3.1-flash-image-preview` (Nano Banana 2)
- Rebranded from "Nano Banana Pro" to "Nano Banana 2" to match Google's naming
- Updated model comparison table with Gemini 3.1 Flash Image as primary recommendation
- Kept `gemini-3-pro-image-preview` as a fallback option

## [1.0.0] - 2025-12-03

### Added
- Initial release of Nano Banana Pro skill
- SKILL.md with MCP setup instructions for both server options
- Prompting guide with SCTD framework and style keywords
- Example prompts for marketing, web design, art, photography, and more
- Character consistency techniques documentation
- Text rendering best practices
- Troubleshooting guide
- Model comparison (Gemini 3 Pro vs 2.5 Flash)

### Features
- Support for Gemini 3 Pro Image (Nano Banana Pro) - highest quality
- Support for Gemini 2.5 Flash Image (Nano Banana) - faster/cheaper option
- 4K resolution output guidance
- Multi-image composition techniques
- Session-based character consistency
